#### inicializar un proyecto en Node

    npm init -y

#### Instalar

    npm install express --save
    npm i ejs
    npm i cors

#### Instalar Resend para enviar mensajes

    npm install resend

#### Pasos para correr proyecto existente

    npm install

    node --watch index.js
